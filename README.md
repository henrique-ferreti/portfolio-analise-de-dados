# Portfólio de Análise de Dados

Este repositório centraliza os principais projetos do meu portfólio em análise de dados, analytics operacional e visualização de informações.

## Objetivo
Demonstrar capacidade de transformar dados em insights por meio de projetos práticos, com foco em lógica analítica, organização, clareza técnica e aplicação em contexto de negócio.

## Projetos

| Projeto | Stack | Objetivo | Link |
|---|---|---|---|
| Análise de Vendas com Python | Python, pandas, matplotlib | Limpeza, agregação e análise de receita | [Abrir](https://github.com/henrique-ferreti/analise-vendas-python) |
| Análise de Pedidos com Java | Java | Processamento de CSV e resumo executivo | [Abrir](https://github.com/henrique-ferreti/analise-pedidos-java) |
| Análise de Campanhas com JavaScript | JavaScript | KPIs de marketing e taxa de conversão | [Abrir](https://github.com/henrique-ferreti/analise-campanhas-javascript) |
| Dashboard Executivo | HTML, CSS, JavaScript | Visualização de indicadores com interface profissional | [Abrir](https://github.com/henrique-ferreti/dashboard-executivo-html-css-js) |
| Insights Operacionais | Node.js | Consolidação e leitura de dados operacionais | [Abrir](https://github.com/henrique-ferreti/insights-operacionais-nodejs) |
| Análise Operacional com SQL | SQL | SLA, atraso, ranking e desempenho operacional | [Abrir](https://github.com/henrique-ferreti/analise-operacional-sql) |

## Competências demonstradas
- Tratamento e organização de dados
- Análise exploratória
- Geração de métricas e KPIs
- Visualização de dados
- Leitura operacional e executiva
- Estruturação de projetos para portfólio profissional
